# SavvyCAN BLE CAN Bridge — Sesja 2026-05-20

## Zrealizowane

### 1. ESP32 — firmware BLE CAN Bridge
- **Plik**: `esp32_firmware/ble_can_bridge/ble_can_bridge.ino`
- MCP2515, 250kbps, kwarc 8MHz
- Serwis BLE CAN: NOTIFY (ESP32→tablet) + WRITE (tablet→ESP32)
- Bufory: 128 Rx / 64 Tx
- Format binarny: ID(4B LE) + DLC(1B) + flags(1B) + data(0-8B)
- Wgrany na ESP32 (MAC b4:bf:e9:09:fa:2a, BLE nazwa "CANBridge")

### 2. SavvyCAN Android — APK zbudowany i zainstalowany
- **APK**: `SavvyCAN_BLE.apk`, pakiet `com.savvycan.android`
- Zakładka **BLE CAN** w ConnectionPage: skanowanie, łączenie, quick-send CAN, log ramek
- `BleManager` (C++/JNI) + `BleHelper` (Java) — obsługa GATT CAN
- Ramki CAN z BLE trafiają do `FrameStore`

### 3. Crash SavvyCAN — naprawiony (3 przyczyny)
- AndroidManifest: nazwy klas Qt 6.10 (`QtActivityBase` zamiast `bindings.QtActivity`)
- build.gradle: usunięty custom → androiddeployqt generuje właściwy
- JDK: wymagane JDK 21 (nie 25) dla Gradle 8.x

### 4. Test BLE LED z komputera — przeszedł pomyślnie
- Połączenie BLE z ESP32 przez bleak (Python)
- ON/OFF/TOGGLE — wszystkie komendy działają, stan potwierdzony odczytem

### 5. Test SLCAN
- SLCAN (WeAct USB2CANV1) skonfigurowany na `slcan0` 250kbps
- Ramki wysyłane, ale MCP2515 ich nie odbiera (TEC=176 REC=128)
- Przyczyna: brak terminacji 120Ω na magistrali CAN

## Budowanie APK (na przyszłość)
```bash
QT=/home/nz2xzhkzfeewkgbu/Qt/6.10.2/android_arm64_v8a
NDK=/home/nz2xzhkzfeewkgbu/Android/Sdk/ndk/28.2.13676358
SDK=/home/nz2xzhkzfeewkgbu/Android/Sdk
JAVA_HOME=/usr/lib/jvm/java-21-openjdk-amd64

cmake -B build_android3 -S android \
  -DCMAKE_TOOLCHAIN_FILE=${QT}/lib/cmake/Qt6/qt.toolchain.cmake \
  -DANDROID_ABI=arm64-v8a -DANDROID_PLATFORM=android-34 \
  -DCMAKE_BUILD_TYPE=Release -GNinja \
  -DCMAKE_PREFIX_PATH=${QT} -DQT_ANDROID_ABIS=arm64-v8a

cmake --build build_android3 --target apk

# Podpisywanie
apksigner sign --ks ~/.android/debug.keystore \
  --ks-pass pass:android --ks-key-alias androiddebugkey --key-pass pass:android \
  build_android3/android-build/build/outputs/apk/release/android-build-release-unsigned.apk

# Instalacja
adb install -r SavvyCAN_BLE.apk
```
